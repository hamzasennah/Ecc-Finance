# ECC Finance

Site officiel statique du club ECC Finance de l'École Centrale Casablanca.

## Sections

- Club: mission, vision, axes opérationnels
- Publications: articles courts et research desk étudiant
- Partenaires: vitrine des collaborations et appel aux partenaires

## Publication GitHub Pages

Le site est prêt pour GitHub Pages via la branche `gh-pages`.

Dans GitHub, activer:

- Settings > Pages
- Source: Deploy from a branch
- Branch: `gh-pages`
- Folder: `/ (root)`

L'URL publique attendue est:

`https://hamzasennah.github.io/Ecc-Finance/`

Les PDFs sources sont volontairement ignorés par Git pour ne pas publier de documents de convention ou de coordonnées personnelles.
