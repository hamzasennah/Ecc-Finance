# ECC Finance

Site officiel statique du club ECC Finance de l'École Centrale Casablanca.

## Sections

- Club: mission, vision, axes opérationnels
- Équipe: bureau élu, pôles et responsabilités
- Rejoindre: formulaire de candidature par pôle
- Programme 2026: ateliers, conférence métiers, workshop et note mensuelle
- Publications: articles courts et research desk étudiant
- Research Desk: mission, méthode, calendrier éditorial et candidature
- Partenaires: page dédiée, packs, processus, formulaire et Partner Pack PDF

## Formulaires

Les formulaires publics utilisent FormSubmit pour envoyer les candidatures et demandes de partenariat vers `club.finance@centrale-casablanca.ma` depuis GitHub Pages, sans backend dédié.

## Publication GitHub Pages

Le site est prêt pour GitHub Pages via la branche `gh-pages`.

Dans GitHub, activer:

- Settings > Pages
- Source: Deploy from a branch
- Branch: `gh-pages`
- Folder: `/ (root)`

L'URL publique attendue est:

`https://hamzasennah.github.io/Ecc-Finance/`

Les PDFs sources sont volontairement ignorés par Git pour ne pas publier de documents de convention ou de coordonnées personnelles.
